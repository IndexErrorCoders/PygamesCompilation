BOMBR
-----

A Bomberman clone. Play against the smack-talking AI over
5 levels of frantic bombing action.


Requires: Python 2.6+, pygame 1.9+, NetworkX 1.8 (included)

To run the game:

    python bombr.py
    

Thanks for playing!
