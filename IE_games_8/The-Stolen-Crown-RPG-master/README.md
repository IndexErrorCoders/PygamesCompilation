The Stolen Crown: A mini-RPG
====================

A fantasy mini-RPG built with Python and Pygame.

Requirements: Python 2.7, Pygame 1.9.1

How to run: python The_Stolen_Crown.py

Video Demo: https://www.youtube.com/watch?v=MkZXaDQfTSo
