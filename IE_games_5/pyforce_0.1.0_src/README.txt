PyForce 0.1.0

AUTHOR:

    Xueqiao Xu <xueqiaoxu@gmail.com>


Introduction:
    PyForce is a Shoot'em up game programmed in Python with the PyGame library.
    Most game contents and the name PyForce are inspired by the mobile game SkyForce.
    
    It's my first game and it has NOT been finished yet.
    Currently only one level is available, and many features are still in progress.
    
    If you have any questions, ideas or suggestions, please feel free to contact me via email.
    

DEPENDENCIES: (NOT Needed if You Downloaded the game in the form of
               Win32 Binary Distribution)

  Python:     http://www.python.org/
  PyGame:     http://www.pygame.org/

  
RUNNING THE GAME:

    On Windows or Mac OS X, locate the "run.py" file and double-click it.

    Othewise open a terminal / console and "cd" to the game directory and run:
    
      python run.pyw

      
CONTROL:

    LEFT/RIGHT/UP/DOWN          control the direction of the plane
    SPACE                       fire
